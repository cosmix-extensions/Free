# MLSBD CloudStream Extension

This is a custom provider extension for CloudStream 3 designed to parse and stream content from MLSBD.

## Features
- Complete catalog of movies and TV series
- Automatic episode batch parsing for series
- Auto extraction of hidden video host links (Savelinks, Luluvid, Hubcloud, etc.)
- Multi-quality selection (480p, 720p, 1080p, 4K)
